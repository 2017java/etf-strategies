# Demo Script（新能源金融需求）

## 目标

在 10 分钟内演示“从模糊需求到可追踪方案”闭环。

## 步骤

1. 展示输入案例：`data/cases/new_energy_finance_case.json`
2. 运行脚本：`python scripts/run_demo.py`
3. 打开输出：
   - `docs/demo/output_bundle.md`
   - `docs/demo/output_bundle.json`
4. 逐段讲解：
   - 需求澄清如何把背景/现状/目标指标结构化
   - PRD 如何覆盖问题、目标、用户故事、范围、验收
   - Plan 如何拆里程碑与执行任务
   - A/B 卡如何定义实验与停启规则
   - 埋点表如何支持后续追踪验证
   - UI/UX 合规清单如何识别风险

## 前后对照建议话术

- 之前：需求表述分散，难以直接执行
- 之后：形成统一交付包，可直接进入评审和开发
