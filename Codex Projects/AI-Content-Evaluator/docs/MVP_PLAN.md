# MVP开发计划

## 第1周：基础框架与数据
- 搭建服务基础结构和API路由
- 实现文本启发式特征：
  - 重复率
  - 模板化短语频率
  - 标点与行文风格一致性
- 实现第一版评分公式与响应结构
- 制定标注规范并准备样本池

## 第2周：图片信号接入
- 增加图片上传与解析流程
- 实现基础取证特征：
  - EXIF元数据可用性
  - 压缩痕迹特征
  - 局部纹理不一致检测
- 融合文本与图片分数，产出统一评分对象

## 第3周：链接分析与可解释性
- 增加链接抓取与正文提取流程
- 引入来源可信度和传播行为特征
- 基于特征贡献生成“证据说明”

## 第4周：评估与阈值策略
- 在1000条标注样本上评估
- 输出准确率、召回率、误报率
- 定义严格/平衡/宽松三档阈值
- 上线管理界面和反馈闭环
