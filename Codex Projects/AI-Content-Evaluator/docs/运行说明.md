# 运行说明

## 1. 创建并激活虚拟环境（Windows PowerShell）
```powershell
cd "D:\AICoding\Codex Projects\AI-Content-Evaluator"
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

## 2. 安装依赖
```powershell
pip install -r requirements.txt
```

## 3. 启动服务
```powershell
uvicorn app.main:app --reload --port 8000
```

## 4. 打开文档
- Swagger UI: http://127.0.0.1:8000/docs
- 健康检查: http://127.0.0.1:8000/health

## 5. 示例请求
### 文本分析
```bash
curl -X POST http://127.0.0.1:8000/analyze/text \
  -H "Content-Type: application/json" \
  -d '{"text":"这是一段待分析的文本内容"}'
```

### 链接分析
```bash
curl -X POST http://127.0.0.1:8000/analyze/url \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com"}'
```

### 图片分析
```bash
curl -X POST http://127.0.0.1:8000/analyze/image \
  -F "file=@test.jpg"
```
