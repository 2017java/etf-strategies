# 💳 信用卡对账系统

一个纯前端的信用卡账单对账和分析Web应用，帮助你自动解析信用卡账单邮件文本，智能分类交易，并提供详细的数据分析和可视化功能。

## ✨ 功能特点

### 核心功能
- **📨 邮件解析** - 自动识别银行邮件格式的账单
- **🧠 智能分类** - 自动分类+手动调整的混合方式
- **📊 数据分析** - 月度趋势、同比/环比分析、分类占比
- **📈 可视化图表** - 饼图、折线图、柱状图
- **📱 响应式设计** - 完美支持桌面端和移动端
- **💾 本地存储** - 所有数据安全存储在浏览器

### 详细功能
- **账单导入**：支持粘贴邮件文本、上传文件
- **交易管理**：查看、筛选、搜索、编辑交易
- **分类管理**：自定义分类，关键词学习
- **对账标记**：标记账单为已对账/未对账
- **数据导出**：JSON格式备份和导入
- **存储统计**：实时查看存储使用情况

## 🚀 快速开始

### 使用方法
1. 下载或克隆这个项目
2. 在浏览器中打开 `index.html`
3. 导航到「账单导入」页面
4. 粘贴银行邮件内容或上传文件
5. 点击「解析账单」并确认导入
6. 查看仪表板和数据分析

### 支持的账单格式
```
2024-01-15  美团外卖       ¥32.50
2024-01-14  滴滴出行       ¥28.00
2024-01-13  淘宝购物       ¥199.00
```

也支持常见银行邮件格式（招商、工商、建设、中国银行等）

## 🎨 技术架构

### 设计模式
- **模块化面向对象设计** - 每个功能独立成类
- **MVC架构思想** - 模型、视图、控制器分离
- **localStorage本地存储** - 数据持久化
- **响应式CSS设计** - 支持多设备

### 核心类库
- `AppController` - 应用控制器，协调整个应用流程
- `EmailParser` - 邮件文本解析器
- `AutoClassifier` - 智能分类器
- `TrendAnalyzer` - 趋势分析器
- `MonthlyBill` - 月度账单模型
- `Transaction` - 交易数据模型

## 📊 功能详解

### 1. 智能分类系统
- **历史分类** - 学习用户手动分类行为
- **规则引擎** - 高优先级自定义规则
- **关键词匹配** - 分类关键词库
- **默认分类** - 兜底策略

### 2. 数据分析功能
- **月度趋势** - 支出/退款时间序列
- **同比分析(YoY)** - 与去年同期比较
- **环比分析(MoM)** - 与上月比较
- **分类占比** - 各分类支出饼图
- **周度分析** - 消费高峰时间

### 3. 统计计算
- 总和、平均值、中位数
- 方差、标准差、变异系数
- 频率分布、累计分布
- 线性回归、相关分析

## 🛠️ 扩展开发

### 添加新的银行解析规则
```javascript
// 在 EmailParser.js 中添加规则
emailParser.addRule({
    name: 'custom_bank',
    pattern: /(\d{4}-\d{2}-\d{2})\s+(.+?)\s+([¥$]?[\d,]+\.?\d*)/,
    groups: { date: 1, description: 2, amount: 3 }
});
```

### 添加自定义分类
```javascript
// 通过UI界面或代码添加
app.addCategory('投资', '📈');
```
